Foodar100 is a self-paced learning module created by Revature with the intention of teaching students what real-world projects can look like. This project is my first solo, real-world project I've attempted and so far I'm proud of it!

So welcome to Foodar, my gourmet food delivery service!

As Unique as You Are!

Provided with just the basics - gray-box outline, website requirements, content - I set out to create my vision of this easy-to-use, ficticious food delivery service. Thus Foodar was born.

And because we are provided a bare-bone structure of the sight, my Foodar is completely unique!


That being said, I've done my fair share of research and all images displayed are CC0, courtesy of Pexels!

But Wait! There's More!

The RevaturePro course doesn't stop once you have the website created and a simple ZIP Code widget activted: The course goes on with intensive lessons on AJAX, NodeJS, and more!

So stick around - more content is on its way!
